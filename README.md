# TimeWeather

A personal **Time/Weather** script for FiveM.  
Allows players to change their own time and weather without affecting others.

---

## Installation

1. Copy the `timeWeather` folder into your `resources` directory.  

2. Make sure **VSync** is removed from your server or any other script that could interfere with this one.

3. Start the script
---

## Features

- Change **time** and **weather** for yourself.  
- Personal, non-synced changes (does not affect other players).  
- Simple commands for easy use.  

---

## Commands

### Weather Commands

- `esunny` – Extra sunny weather  
- `sunny` – Regular sunny weather  
- `rain` – Rainy weather  
- `xmas` – Snowy weather  

### Time Commands

- `day` – Set daytime  
- `night` – Set nighttime  
- `morning` – Set morning  
- `evening` – Set evening
